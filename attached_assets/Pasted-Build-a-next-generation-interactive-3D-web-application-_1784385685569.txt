Build a next-generation interactive 3D web application where a complete house is generated from millions of animated particles.

Project Name:
Particle House Builder

Goal:
Create a futuristic visualization where a house is assembled from glowing particles in real time. Users can interact with every part of the structure using hand gestures, mouse movement, or touch.

Features
------------

1. House Construction Animation

Initially there is only empty space.

Millions of golden particles float in the air.

Particles slowly gather together.

Foundation appears.

Walls are created.

Windows assemble.

Roof forms.

Furniture appears.

Garden grows.

Everything is built using particles.

2. Interactive Building

Rebuild it.
